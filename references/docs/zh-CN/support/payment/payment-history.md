# 查看您的付款历史记录

> 访问和查看您过去的付款和发票

您可以通过帐户仪表板查看完整的付款历史记录并下载发票。

## 访问付款历史记录

1. 登录您的 ComfyUI 帐户并打开个人菜单
2. 点击**用户设置**
3. 在设置面板中选择**积分**
4. 点击**发票历史记录**进入 Stripe 账单门户，即可看到对应的支付历史记录
