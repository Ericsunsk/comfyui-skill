# 订阅 Comfy Cloud

> 了解如何订阅 Comfy Cloud

## 如何订阅

您可以正常浏览和使用 Comfy Cloud,但需要订阅才能运行工作流。

订阅 Comfy Cloud 的步骤:

1. 访问 [cloud.comfy.org](https://cloud.comfy.org/?utm_source=docs) 并登录您的账户。
2. 点击**订阅**按钮。

3. 选择您想要的订阅计划。

有关 Comfy Cloud 订阅计划和定价的详细信息,请访问 [comfy.org/cloud/pricing](https://www.comfy.org/cloud/pricing)。

Comfy Cloud 订阅通过我们的支付处理商 Stripe 进行管理。

## 订阅续订

您的订阅会在每月与首次订阅相同的日期和时间自动续订。

除非您设置取消计划,否则所有 Comfy Cloud 订阅计划都会自动续订。
