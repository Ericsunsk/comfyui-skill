# 管理您的 Comfy Cloud 订阅

> 了解如何管理您的 Comfy Cloud 订阅

## 访问订阅管理

要管理您的 Comfy Cloud 订阅:

1. 访问并登录 [cloud.comfy.org](https://cloud.comfy.org/?utm_source=docs)
2. 导航到设置菜单
3. 进入 **Plan & Credits**(计划与积分)

4. 点击 **Manage subscription**(管理订阅)

这将带您进入 Stripe,您可以在那里管理订阅详情。
