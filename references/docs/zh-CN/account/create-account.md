# 创建 Comfy 账户

> 了解如何创建新的 Comfy 账户以访问 ComfyUI 的所有功能和服务。

您的 Comfy 账户可让您访问[合作伙伴节点（API 节点）](/zh-CN/tutorials/partner-nodes/overview)和[云订阅](https://www.comfy.org/cloud)，使您能够在 ComfyUI 平台上使用高级功能和服务。

## 在 Comfy Cloud 上创建 Comfy 账户

您可以直接在 Comfy Cloud 上创建 Comfy 账户以使用 ComfyUI：

1. 访问 [Comfy Cloud](https://www.comfy.org)
2. 点击**注册**或**创建账户**
3. 选择以下登录方式之一：
   * **电子邮件**：输入您的电子邮件地址并创建密码
   * **Google**：使用您的 Google 账户注册
   * **GitHub**：使用您的 GitHub 账户注册
4. 完成注册流程
5. 如果使用电子邮件注册，请验证您的电子邮件

## 在本地创建 Comfy 账户

如果您已在本地安装 ComfyUI，可以通过应用程序创建 Comfy 账户：

1. 在本地计算机上打开 ComfyUI
2. 在界面中导航到**设置**
3. 进入**用户**部分（详见[用户设置](/zh-CN/interface/user)）
4. 点击**创建账户**或**注册**
5. 选择以下登录方式之一：
   * **电子邮件**：输入您的电子邮件地址并创建密码
   * **Google**：使用您的 Google 账户注册
   * **GitHub**：使用您的 GitHub 账户注册
6. 完成注册流程

## 后续步骤

账户创建并验证后：

* [登录您的账户](/zh-CN/account/login)
* 设置您的个人资料偏好
* 开始使用 ComfyUI 功能
* 探索教程和文档

## 故障排除

如果您在创建账户时遇到问题：

* 确保您的电子邮件地址有效且尚未注册
* 检查您的密码是否符合最低要求
* 清除浏览器缓存并重试
* 如果问题仍然存在，请联系[支持](/zh-CN/support/contact-support)
